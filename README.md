# Fractol

![Fract'ol](AlarmedThriftyDikkops-max-1mb.gif)

This project is about creating graphically beautiful fractals.
Note that **macOS is only supported**.

## Installation
1. clone the project:
  ``` 
  https://github.com/Xperaz/Fractol.git 
  ```
2. Make
  ``` 
    make 
  ```
3. Run the program
  ```
    ./fractol mandelbrot
  ```
you can run : 
- mandlebrot
- julia
- burningship
